3. How JavaScript event loop works?

→ The Event Loop has one simple job to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.

4. How many ways you can get undefined?

    → 8 ways of getting undefined:
  1. Variable that is not initialize.
  2. If don't return anything from function
  3. Parameter that is not passed in a function will be undefined
  4. If return has nothing on the right side will return undefined
  5. Property that doesn't exist on an object will give you undefined
  6. Access array elements outside of the array range.
  7. Deleting an element inside an array.
  8. Set a value directly to undefined.
